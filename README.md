# 📘 Smart Phone Book

A modern and responsive **Smart Phone Book** web application built using **HTML, CSS, and JavaScript**. The application allows users to manage contacts efficiently with features such as adding, updating, deleting, searching, marking favorites, tracking call history, and storing data permanently using the browser's Local Storage.

---

## ✨ Features

- 📇 View all contacts
- ➕ Add new contacts
- ✏️ Update existing contacts
- ❌ Delete contacts
- 🔍 Live search by name or phone number
- ⭐ Mark and unmark favorite contacts
- 📞 Simulate phone calls
- 📜 Maintain call history with timestamps
- 📈 View frequently called contacts
- 🔠 Contacts grouped alphabetically
- 💾 Automatic data persistence using Local Storage
- 📱 Responsive user interface

---

## 📚 Data Structures & Algorithms

This project demonstrates the practical implementation of:

- **Binary Search Tree (BST)**
  - Stores contacts alphabetically.
  - Supports efficient insertion, searching, updating, and deletion.

- **In-order Traversal**
  - Displays contacts in sorted alphabetical order.

- **Fuzzy Search**
  - Allows flexible searching by matching partial names.

- **Call Frequency Tracking**
  - Maintains the number of calls made to each contact.

- **Local Storage**
  - Stores contacts, favorites, and call history permanently in the browser.

## 💡 How It Works

1. Contacts are stored inside a Binary Search Tree.
2. The BST keeps contacts sorted alphabetically.
3. All contacts are saved to Local Storage.
4. Every phone call updates the call frequency and call history.
5. Favorites are stored separately for quick access.
6. Search uses a fuzzy matching algorithm for improved user experience.
